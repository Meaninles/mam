# 统一文件管理系统-客户端任务中心传输任务子页面设计方案

## 文档说明

- 更新时间：2026-04-11
- 对照代码：`client/src/pages/TaskCenterPage.tsx`、`client/src/pages/TaskCenterWorkspace.tsx`、`client/src/lib/jobsApi.ts`
- 范围：仅描述当前任务中心中“传输任务”子页的真实实现

## 1. 页面定位

传输任务子页是任务中心中承接运行期管理的主子页，用于统一展示：

- 导入任务
- 同步任务

它负责的不是任务创建前的编排，而是创建后的跟踪、筛选、操作与跳转。

## 2. 当前与顶层多标签工作区关系

### 2.1 入口

当前可以从以下位置进入并聚焦传输任务子页：

- 左侧主导航 -> 任务中心
- 导入中心提交后自动跳入
- 通知中心跳入
- 文件中心删除冲突弹层中的“查看任务”

### 2.2 工作区行为

任务中心作为顶层标签页运行，当前支持：

- 保活
- 刷新
- 被通知 / 文件中心带上下文聚焦

传输任务子页只是任务中心内部的一个 tab，不是顶层标签。

## 3. 当前页面结构

传输任务子页当前包含：

- 子页切换层：传输任务 / 其他任务
- 工具条层
- 选中态批量工具条
- 任务列表层
- 任务详情抽屉
- 子项详情抽屉

## 4. 当前工具条能力

### 4.1 业务类型筛选

当前支持：

- 导入
- 同步

### 4.2 链路类型筛选

当前支持：

- 全部
- 复制
- 上传
- 下载

### 4.3 状态筛选

当前支持：

- 全部
- 活跃中
- 运行中
- 等待确认
- 暂停任务
- 异常待处理
- 部分成功
- 失败
- 已完成
- 已取消

### 4.4 资产库筛选

当前支持按资产库过滤。

### 4.5 搜索

当前会搜索：

- 任务标题
- 源端 / 目标端
- 源路径 / 目标路径
- 子项文件名与路径

### 4.6 排序

当前支持：

- 默认排序
- 优先级
- 进度
- 名称

### 4.7 分页

当前支持：

- 10 / 20 / 50 / 100 条每页

## 5. 当前列表设计

### 5.1 一级任务行

每个一级任务当前展示：

- 勾选框
- 展开按钮（若存在子项）
- 动态标题
- 路径摘要
- 文件 / 文件夹摘要
- 任务标签组
- 进度条与进度值
- 源端 -> 目标端元信息
- 当前阶段
- 当前速度
- 剩余时间
- 状态 pill
- 最近异常摘要 / 结果摘要
- 操作按钮

### 5.2 标题生成规则

当前标题不是固定使用 `task.title`，而是会根据任务类型动态推导：

- 单目录任务优先显示目录名
- 多文件任务优先汇总文件名
- 导入任务会优先用子项名称聚合

### 5.3 二级子项区

展开后可看到文件级子项，当前展示：

- 勾选框
- 子项名称
- 路径
- 大小
- 优先级
- 进度
- 当前阶段
- 速度
- 状态
- 子项操作按钮

## 6. 当前异常提示

当前传输任务页已重新接回异常摘要、异常浮窗和异常中心正式跳转。

- 任务列表与详情会显示关联异常数量和最近异常摘要
- 可按单条异常跳转异常中心
- 可按当前任务跳转异常中心并自动带任务筛选

## 7. 当前操作设计

### 7.1 主任务操作

主任务当前支持的按钮组合取决于状态，实际可出现：

- 详情
- 暂停
- 继续
- 重试
- 取消

### 7.2 子项操作

子项当前支持：

- 详情
- 暂停
- 继续
- 取消

### 7.3 批量操作

当前支持对选中任务或子项做批量状态操作与优先级调整，具体能力受当前状态控制。

## 8. 当前详情抽屉

### 8.1 主任务详情抽屉

当前展示：

- 业务类型
- 链路类型
- 状态
- 优先级
- 源端 / 目标端
- 源路径 / 目标路径
- 文件数量 / 文件夹数量
- 当前速度
- 剩余时间
- 执行引擎
- 外部任务 ID
- 外部状态
- 失败位置

- 暂停 / 继续 / 重试 / 取消
- 设为高优先级 / 恢复普通优先级
- 查看文件中心

### 8.2 子项详情抽屉

当前展示：

- 节点类型
- 状态
- 优先级
- 当前阶段
- 大小
- 速度
- 进度
- 执行引擎
- 外部任务 ID
- 外部状态
- 失败位置
- 原始错误
- 源路径
- 目标路径
- 异常数量

## 9. 当前数据来源

传输任务子页数据当前来自任务中心工作区的真实后端消费层：

- `TaskCenterWorkspace`
- `jobsApi.list`
- `jobsApi.detail`
- `jobsApi.subscribe`
- `jobsApi.pause` / `resume` / `cancel` / `retry`
- `jobsApi.pauseItem` / `resumeItem` / `cancelItem`
- `jobsApi.updatePriority`

页面不再以 `PersistedState.taskRecords / taskItemRecords` 作为正式数据源。

## 10. 当前与其他页面的联动

### 10.1 与导入中心

- 导入中心提交导入后，会创建传输任务并切到本页

### 10.2 与文件中心

- 任务详情可定位到文件中心
- 文件中心同步动作会创建传输任务

### 10.3 与通知中心

- 通知可直接把任务中心切到特定任务

## 11. 当前实现边界

以下能力当前没有实现：

- 任务日志流
- 子项级重试按钮
- 可配置列
- 以任务页直接发起“测试探针式外部执行器自检”

## 12. 当前版本结论

截至 2026-04-12，传输任务子页已经切到真实作业消费层：

- 业务类型、链路类型、状态、资产库多维过滤
- 一级任务与二级子项结构
- 详情抽屉、批量操作、主任务与子任务控制
- SSE 实时刷新
- 文件中心切片 7 的同步到端点现在会创建真实 `REPLICATE` 作业，并在本页稳定展示
- `CD2_REMOTE_UPLOAD` 与 `ARIA2` 的外部状态已纳入任务阶段文案、详情抽屉和失败定位信息
- CloudDrive2 上传已按两阶段语义展示：
  - 写入缓存阶段不会被当作“上传成功”
  - 只有外部状态进入 `Finish`，才表示真正完成云端上传

当前应按“真实作业消费页”来描述；其中云端上传、云端下载的可观测性字段已进入正式展示，重点已转为继续补齐更高强度的真实链路验证与控制回归。
