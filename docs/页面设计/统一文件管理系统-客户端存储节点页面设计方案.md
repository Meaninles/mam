# 统一文件管理系统-客户端存储节点页面设计方案

## 文档信息

- 文档名称：统一文件管理系统-客户端存储节点页面设计方案
- 文档用途：记录当前客户端“存储节点”页面的实际实现
- 适用范围：Windows 桌面客户端首版原型
- 当前状态：已按当前实现刷新
- 更新时间：2026-04-08

## 1. 页面定位

当前“存储节点”不是单纯的状态看板，而是一个存储配置工作台。页面围绕“挂载文件夹”展开，并补充 NAS 与网盘配置。

页面重点完成：

- 管理挂载文件夹
- 管理 NAS 主机
- 管理网盘接入
- 发起扫描与连接测试
- 查看扫描历史

## 2. 页面入口

- 存储节点是左侧导航一级页面
- 已纳入顶层多标签工作区

## 3. 当前子页结构

页面当前分为 3 个子页：

1. 挂载管理
2. NAS 管理
3. 网盘管理

## 4. 顶部工具区

当前顶部支持：

- 子页切换
- 类型筛选
- 状态筛选
- 搜索
- 新增挂载文件夹 / 新增 NAS / 新增网盘

搜索提示会随子页变化：

- 挂载页：搜索名称、路径、资产库
- NAS 页：搜索名称、地址、账号
- 网盘页：搜索名称、目录、账号别名

## 5. 挂载管理

### 5.1 列表展示

当前表格展示：

- 勾选框
- 名称
- 资产库
- 类型
- 路径
- 挂载模式
- 扫描状态
- 心跳周期
- 容量
- 鉴权状态
- 风险标签
- 操作

### 5.2 批量动作

当前支持：

- 批量扫描
- 批量连接测试
- 清空选择

### 5.3 单项动作

当前支持：

- 编辑
- 删除
- 连接测试
- 扫描
- 设置心跳
- 查看扫描历史
- 打开相关异常中心
- 打开相关任务中心

### 5.4 新增 / 编辑挂载

当前挂载表单支持三类来源：

- 本地
- NAS
- 网盘

字段按类型动态变化：

- 本地：本地目录
- NAS：NAS 来源、NAS 共享文件夹
- 网盘：网盘来源、网盘文件夹名 / ID

通用字段包括：

- 挂载名称
- 所属资产库
- 挂载模式
- 心跳周期

## 8. 当前弹层

当前页面包含以下弹层：

- 新增 / 编辑挂载文件夹
- 新增 / 编辑 NAS
- 新增 / 编辑网盘
- 设置心跳
- 扫描历史
- 连接测试结果

## 9. 当前数据来源

当前页面使用 `storageNodesApi`，具备双路径：

- 浏览器 fallback mock
- Tauri Rust 命令 mock

如果运行在 Tauri 环境，会调用本地命令：

- 加载仪表盘
- 保存配置
- 扫描
- 连接测试
- 更新心跳
- 读取历史
- 删除
- 浏览本地目录

## 10. 当前与其它页面的关系

- 扫描任务会进入任务中心 > 其他任务
- 挂载异常可跳转异常中心
- 目标端查看可回到导入中心或文件中心上下文

## 11. 当前边界

- 没有真实磁盘扫描
- 没有真实 NAS / 网盘鉴权
- 没有真实挂载
- 连接测试结果为 mock

## 12. 当前结论

截至 2026-04-08，存储节点页已经形成结构完整的配置工作台，是当前客户端唯一同时打通浏览器 mock 与 Tauri mock 的业务页面。
