# 统一文件管理系统-客户端存储节点页面设计方案

## 文档信息

- 文档名称：统一文件管理系统-客户端存储节点页面设计方案
- 文档用途：用于客户端“存储节点”页面的产品、交互与界面实现对齐
- 适用范围：Windows 桌面客户端首版
- 当前状态：已按当前客户端实现更新
- 相关页面：客户端“文件中心”另见 `docs/页面设计/统一文件管理系统-客户端文件中心页面设计方案.md`

## 1. 页面定位

当前客户端中的“存储节点”不再是单一节点列表页，而是客户端侧的存储配置工作台。页面围绕“资产库挂载目录管理”展开，同时补充 NAS 主机配置与网盘配置两个子页。

页面目标不是做成监控大盘，而是让用户完成以下日常管理动作：

- 管理各资产库实际挂载的具体目录
- 管理 NAS 主机配置
- 管理网盘接入配置
- 对挂载目录发起扫描与连接测试
- 查看扫描历史与相关任务 / 异常入口；当前扫描任务会统一进入“任务中心 > 其它任务”子页面跟踪

## 2. 当前实现总览

### 2.1 子页结构

客户端存储节点页面当前拆分为 3 个子页：

1. `挂载文件夹管理`
2. `NAS 管理`
3. `网盘管理`

其中：

- `挂载文件夹管理` 是主工作区，展示的是“某个资产库挂载到哪个本地目录 / NAS 子目录 / 网盘目录”
- `NAS 管理` 展示 NAS 主机配置列表
- `网盘管理` 展示网盘接入配置列表

### 2.2 顶部布局

页面顶部采用三段式轻工具布局：

1. 页头：左侧显示行为反馈文字，右侧为导入中心与通知入口
2. 子页切换条：用于切换 3 个子页
3. 工具条：用于筛选、搜索和新增

说明：

- 行为反馈只在用户触发扫描、连接测试、保存、删除等动作后显示
- 反馈以纯文字形式贴近页头左侧，不使用额外卡片或横幅容器
- 深浅色主题切换入口已从页头移除，仅在设置页的“外观”中配置

## 3. 子页职责

### 3.1 挂载文件夹管理

负责管理资产库实际使用的挂载目录记录。

每条记录包含：

- 挂载名称
- 所属资产库
- 文件夹类型：本地 / NAS / 网盘
- 实际挂载路径
- 扫描状态
- 最近扫描时间
- 心跳周期
- 容量 / 可用空间摘要

补充规则：

- 一个挂载文件夹记录只允许绑定一个资产库
- 列表路径下不再展示“已绑定 X 个资产库”
- 支持多选，但当前只保留批量扫描、批量连接测试与清空选择
- 不提供批量设置心跳

### 3.2 NAS 管理

负责管理 NAS 主机级配置。

列表列定义：

- 名称
- 地址
- 账号密码摘要
- 鉴权状态
- 操作

说明：

- 鉴权状态以最后一次连接测试结果为准
- 操作区当前包含：连接测试、编辑、更多

### 3.3 网盘管理

负责管理网盘接入配置。

列表列定义：

- 名称
- 网盘类型
- 接入方式
- 挂载目录
- 鉴权状态
- 操作

说明：

- 当前网盘类型仅实现 `115`
- 鉴权状态以最后一次连接测试结果为准
- 操作区当前包含：连接测试、编辑、更多

## 4. 列表与分页

### 4.1 分页规则

三个子页当前均支持分页显示。

分页规则如下：

- 支持页码切换
- 支持首页 / 上一页 / 下一页 / 末页
- 支持设置每页数量：`10 / 20 / 50 / 100`
- 默认每页数量：`20`

### 4.2 分页样式原则

- 分页器放在列表底部
- 视觉风格延续当前客户端按钮、圆角、边框与颜色体系
- 不引入与客户端不一致的第三方分页样式

### 4.3 当前行高策略

当前实现中已将存储列表行高压缩为更紧凑的密度：

- 表头字号略小于正文
- 行内上下 padding 缩小
- 标签、状态胶囊与摘要文字的垂直间距收紧

目标是在可读的前提下提高单位屏幕的信息承载量。

## 5. 行内操作

### 5.1 挂载文件夹管理

挂载文件夹列表右侧操作顺序为：

1. `连接测试`
2. `立即扫描`
3. `编辑`
4. `更多`

`更多` 菜单当前包含：

- 设置心跳
- 查看扫描历史
- 查看相关任务（跳转到“任务中心 > 其它任务”并查看对应扫描任务；目标页当前为单层任务列表，不在主列表内展开扫描子项）
- 查看相关异常
- 删除

### 5.2 NAS 管理 / 网盘管理

NAS 与网盘列表右侧操作顺序为：

1. `连接测试`
2. `编辑`
3. `更多`

`更多` 菜单当前仅保留低频操作，如删除。

### 5.3 Tooltip 规则

所有图标按钮均使用客户端内统一的自定义 tooltip。

说明：

- 不再依赖浏览器原生 `title`
- 鼠标连续从一个图标移动到下一个图标时，tooltip 应尽量稳定显示
- tooltip 文案只保留简短动作名称，例如 `连接测试`、`立即扫描`、`编辑`、`更多操作`

## 6. 菜单与弹层

### 6.1 更多操作菜单

当前实现中，“更多操作”菜单支持以下关闭方式：

- 再次点击 `更多操作` 按钮
- 点击页面任意空白区域
- 执行菜单项后自动关闭

### 6.2 弹层类型

当前页面使用的隐藏式弹层包括：

- 新增 / 编辑挂载文件夹抽屉
- 新增 / 编辑 NAS 抽屉
- 新增 / 编辑网盘抽屉
- 连接测试结果弹窗
- 扫描历史弹窗
- 心跳设置弹窗

## 7. 新增与编辑交互

### 7.1 新增挂载文件夹

新增挂载文件夹时，表单必须先选择：

- 所属资产库
- 文件夹类型：本地 / NAS / 网盘

然后根据类型展示不同字段：

#### 本地

- 挂载名称
- 所属资产库
- 本地目录
- 挂载模式
- 心跳周期
- 备注

说明：

- 本地目录通过 Windows 原生文件夹选择器获取
- 不要求手输本地路径作为主要方式

#### NAS

- 挂载名称
- 所属资产库
- NAS 来源（从已配置 NAS 中选择）
- NAS 共享文件夹
- 挂载模式
- 心跳周期
- 备注

#### 网盘

- 挂载名称
- 所属资产库
- 网盘来源（从已配置网盘中选择）
- 网盘文件夹名 / ID
- 挂载模式
- 心跳周期
- 备注

### 7.2 新增 / 编辑 NAS

NAS 配置抽屉当前包含：

- 名称
- 地址
- 账号
- 密码
- 备注

### 7.3 新增 / 编辑网盘

网盘配置抽屉当前包含：

- 名称
- 网盘类型（当前固定为 `115`）
- 接入方式：`填入 Token` / `扫码登录获取 Token`
- 扫码登录类型：`微信小程序` / `支付宝小程序` / `电视端`
- 账号别名
- 挂载目录
- Token
- 备注

说明：

- 当前仍允许在表单中手动填写 Token
- 扫码登录为客户端 mock 交互，生成会话后回填 Token

## 8. 状态定义

### 8.1 挂载文件夹

挂载文件夹列表当前支持以下状态表达：

- 最近扫描成功
- 最近扫描失败
- 等待队列
- 扫描中
- 可用
- 异常
- 已停用

### 8.2 NAS / 网盘

NAS 与网盘列表当前的“鉴权状态”来自最后一次连接测试结果。

当前客户端展示语义为：

- `鉴权正常`
- `鉴权异常`
- `尚未测试`

## 9. 与通知、设置的联动

### 9.1 通知中心

移动硬盘 / U 盘接入提醒不在存储节点页面内常驻展示。

当前策略：

- 接入提醒先进入通知中心
- 用户可标记已读
- 标记已读后页头通知红点消失，但通知记录仍保留

### 9.2 外观设置

深浅色主题切换入口不再位于页头。

当前策略：

- 页头只保留导入中心与通知
- 外观主题统一在设置页的“外观”中配置

## 10. 当前实现结论

当前客户端存储节点页面的实际方向已明确为：

- 以 `挂载文件夹管理` 作为存储节点主工作区
- 以 `NAS 管理` 和 `网盘管理` 承载来源配置
- 以轻工具条 + 列表 + 抽屉 / 弹窗作为主要交互模型
- 以连接测试、扫描、挂载配置为核心能力
- 以分页与紧凑列表支持更高密度的运维视图

该实现已经不再对应“单一存储节点总表页”的早期方案，后续文档与开发应以本版本结构为准。
