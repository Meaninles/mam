# 统一文件管理系统-客户端任务中心其他任务子页面设计方案

## 文档说明

- 更新时间：2026-04-11
- 对照代码：`client/src/pages/TaskCenterPage.tsx`、`client/src/pages/TaskCenterWorkspace.tsx`、`client/src/lib/jobsApi.ts`
- 范围：仅描述当前任务中心中“其他任务”子页的真实实现

## 1. 页面定位

其他任务子页用于承接不属于传输链路的后台任务，当前实际覆盖：

- 扫描
- 元数据解析
- 校验
- 删除清理

它和传输任务子页共用一套页面骨架，但筛选维度和字段重点不同。

## 2. 当前与顶层多标签工作区关系

其他任务子页属于任务中心内部 tab，不是独立顶层标签。进入方式包括：

- 左侧主导航 -> 任务中心后手动切到“其他任务”
- 存储节点页面中的“查看相关任务”
- 通知中心跳到任务中心

## 3. 当前页面结构

其他任务子页当前包含：

- 子页切换层
- 工具条层
- 选中态批量工具条
- 一级任务列表
- 最近异常摘要区
- 详情抽屉

## 4. 当前筛选与搜索

### 4.1 任务类型筛选

当前支持：

- 全部
- 扫描
- 元数据解析
- 校验
- 删除清理

### 4.2 状态筛选

当前支持：

- 全部
- 活跃中
- 待执行
- 运行中
- 等待确认
- 异常待处理
- 等待清理
- 暂停任务
- 部分成功
- 失败
- 已完成
- 已取消

### 4.3 搜索

当前搜索会命中：

- 任务标题
- 范围摘要
- 源路径 / 目标路径
- 结果摘要
- 当前阶段
- 子项名称与路径
- 关联异常文案

### 4.4 排序

当前支持：

- 默认排序
- 优先级
- 进度
- 名称

### 4.5 分页

与传输任务子页一致，支持：

- 10 / 20 / 50 / 100 条每页

## 5. 当前列表设计

### 5.1 一级任务行

当前一级任务行会展示：

- 勾选框
- 任务标题
- 路径或范围摘要
- 文件 / 对象数量
- 当前阶段
- 进度条
- 当前速度
- 状态 pill
- 最近异常摘要
- 行操作

### 5.2 摘要规则

当前其他任务行会优先展示：

- `scopeLabel`
- `sourcePath`
- `targetPath`
- `resultSummary`

不同类型任务的摘要风格不同，但仍复用统一列表样式。

### 5.3 删除清理任务的特殊口径

删除清理任务当前属于 `otherTaskType = DELETE_CLEANUP`。在列表中：

- 会被归入“其他任务”
- 会显示待处理数量或清理进度
- 通常来自文件中心删除动作或种子数据

## 6. 当前异常提示

当前其他任务页已不再保留任务中心内的异常浮窗和异常中心正式跳转。

- 任务详情里仍可展示最近异常摘要或结果摘要
- 异常相关能力当前只保留接口化占位，后续再由异常域接管

## 7. 当前操作设计

### 7.1 主任务操作

根据状态，当前可出现的按钮有：

- 详情
- 暂停
- 继续
- 重试
- 取消

### 7.2 批量操作

当前也支持对选中其他任务进行批量状态修改和优先级调整。

### 7.3 详情跳转

详情抽屉底部按钮会根据任务类型变化：

- 扫描类任务优先提供“查看存储节点”
- 其它类型优先提供“查看文件中心”

## 8. 当前详情抽屉

### 8.1 展示内容

其他任务详情抽屉当前展示：

- 业务类型
- 状态
- 当前阶段
- 优先级
- 源端 / 目标端
- 源路径 / 目标路径
- 文件数量 / 文件夹数量
- 当前速度
- 剩余时间
- 范围摘要
- 结果摘要
- 创建时间
- 最近更新
- 异常数量
- 最近异常

### 8.2 子项列表

如果任务有子项，还会在详情抽屉中展示：

- 子项名称
- 路径或结果标签
- 阶段
- 状态
- 进度条

## 9. 当前数据来源

其他任务子页和传输任务子页一样，当前都由任务中心工作区消费真实后端作业数据：

- `TaskCenterWorkspace`
- `jobsApi.list`
- `jobsApi.detail`
- `jobsApi.subscribe`
- `jobsApi.pause` / `resume` / `cancel` / `retry`
- `jobsApi.pauseItem` / `jobsApi.resumeItem` / `jobsApi.cancelItem`
- `jobsApi.updatePriority`

## 10. 当前与其他页面的联动

### 10.1 与存储节点

- 扫描类任务详情支持跳转存储节点
- 存储节点页面也可反向跳入任务中心

### 10.2 与文件中心

- 删除清理与部分校验类任务可跳到文件中心
- 文件中心删除动作会创建删除清理任务

## 11. 当前实现边界

以下内容当前没有实现：

- 独立的其他任务专属页面
- 类型化详情页
- 真实后台执行器日志
- 任务配置编辑
- 子项级重试
- 可配置列
- 任务中心内的异常中心正式跳转

## 12. 当前版本结论

截至 2026-04-12，其他任务子页已经切到真实作业消费层：

- 继承任务中心统一骨架
- 重点覆盖扫描、解析、校验、删除清理
- 具备状态管理、详情抽屉、主任务与子任务控制
- SSE 实时刷新
- 文件中心切片 7 的删除端点副本、删除资产真实作业会在本页展示为删除清理类任务

当前应按“真实作业消费页”描述它；但异常域真实闭环仍未完成，任务中心内不再提供正式的异常中心跳转。
